CSS は HTML で書かれた文書の表示スタイルを定義する。CSS を HTML ファイルに適用するには link を指定すればいい。二つ以上の CSS ファイルを同じページに適用することもできる。その場合は link を続けて書く。CSS のメディアクエリを使うと、レスポンシブなページを作成することも可能である。

CSS はセレクタとスタイルを組み合わせて書く。

各スタイルの適用範囲は細かく指定することができる。ページ内の唯一の要素を指定するときは id 属性を、二つ以上の要素をまとめて指定するときは class 属性を設定する。セレクタは、複数セレクタや小孫セレクタで組み合わせることができる。

スタイルを指定する際は、ボックスモデルや display プロパティ、あるいは引継ぎやスタイル適用の優先順位を考慮する。すべての要素はボックスをもち、内から順にコンテンツ、パディング、ボーダー、マージンの四層構造になっている。要素は、display プロパティの値により、ブロックレベル要素とインライン要素とに分類される。文字のスタイルは親要素から子要素へ引き継がれるし、通常、引き継がれないスタイルもプロパティの値に inherit と書くと強制的に親要素のスタイルを引き継げる。そして、スタイルの適用順位はその定義される場所およびセレクタの詳細度で決まる。
