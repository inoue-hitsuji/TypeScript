**HTML** は、**マークアップ**によって文書構造を定義する。マークアップとは、テキストにタグや記号を追加して、文書の構造や意味を表現することである。HTML では、テキストを**タグ**で囲むことで、段落、見出し、強調などの構造や意味を定義する。定義することによって、検索エンジンや JS、CSS が文書を解析できるようになる。

タグで囲まれた部分を**要素内容**といい、タグと要素内容を合わせて**要素**という。要素内容の前にあるタグを開始タグ、後にあるタグを終了タグと呼ぶ。要素内容をもたない要素を**空要素**といい、空要素は開始タグのみ書く。また、開始タグの中に、要素の性質を表す**属性**を書くことができる。

例：`<p>テキスト</p>` では、`<p>` が開始タグ、`テキスト` が要素内容、`</p>` が終了タグである。`<img src="image.jpg">` は空要素の例である。

HTML の 1 行目には**文書型宣言（DOCTYPE宣言）**を書き、このページが HTML で書かれていることを表す。HTML の 2 行目以降は html タグで全体を囲み、html の中に **head** と **body** を書く。head は文書情報を、body はブラウザの画面に表示する内容を表す。**文書情報**というのは、その文書に関するメタ情報のことである。

例：
```html
<!DOCTYPE html>
<html>
  <head>
    <title>ページタイトル</title>
  </head>
  <body>
    <p>本文</p>
  </body>
</html>
```

すべてのタグは 7 つのカテゴリーに分類され、その配置ルールは**コンテンツモデル**で定められている。

- メタデータコンテンツ
    > head 要素内に配置される
    - `<title>`：ページタイトル
    - `<meta>`：メタ情報
    - `<link>`：外部リソースへのリンク
    - `<style>`：CSS
    - `<script>`：JavaScript
- フローコンテンツ
    > body 要素内に配置される
    - `<div>`：汎用コンテナ
    - `<p>`：段落
    - `<ul>`, `<ol>`, `<li>`：リスト
    - `<blockquote>`：引用
    - セクショニングコンテンツ
        - `<article>`：独立した記事
        - `<section>`：セクション
        - `<nav>`：ナビゲーション
        - `<aside>`：補足情報
    - ヘッディングコンテンツ
        - `<h1>`〜`<h6>`：見出し
    - フレージングコンテンツ
        - `<span>`：インラインコンテナ
        - `<a>`：リンク
        - `<strong>`：強調
        - `<em>`：強調（斜体）
        - `<code>`：コード
    - エンベディッドコンテンツ
        - `<img>`：画像
        - `<audio>`：音声
        - `<video>`：動画
        - `<iframe>`：埋め込みフレーム
    - インタラクティブコンテンツ
        - `<a>`：リンク
        - `<button>`：ボタン
        - `<input>`：入力欄
        - `<select>`：選択リスト
        - `<textarea>`：テキストエリア
