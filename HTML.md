HTML は、ページの構成要素をタグで囲むことによって文書構造を定義する。定義することによって、検索エンジンや JS、CSS がこれを解析できるようになる。

タグで囲まれた部分を要素内容といい、タグと要素内容を合わせて要素という。要素内容の前に書くタグを開始タグ、後に書くタグを終了タグと呼ぶ。要素内容をもたない要素を空要素といい、空要素は開始タグのみ書く。また、開始タグの中に、要素の性質を表す属性を書くことができる。

HTML の 1 行目には文書型宣言を書き、このページが HTML で書かれていることを表す。HTML の 2 行目以降は html タグで全体を囲み、html の中に head と body を書く。head は文書情報を、body はブラウザの画面に表示する内容を表す。文書情報というのは、その文書に関するメタ情報のことである。

すべてのタグは 7 つのカテゴリーに分類され、その配置ルールはコンテンツモデルで定められている。
