- 事実を観察し、記録したものをデータという
- TypeScript が扱うデータを値という
- 値の分類が型である
- TypeScript の扱う型には、プリミティブとオブジェクトがある
- プリミティブは下記 7 種ある
    - number
    - bigint
    - string
    - boolean
    - null
    - undefined
    - symbol
- プリミティブ以外をオブジェクトという
- プリミティブを組み合わせるとオブジェクトとなる
    - つまり、オブジェクトは値を複数持つ

- 値の表現が式である
- 式は評価すると値を返す
    - 「評価する」ことを「計算する」ともいう
        - 計算は、等価性による値の書き換えである
    - 式にはリテラルと演算子がある
- 変数は、値を記憶する場所である
- 変数名は変数に付ける名前である
- オブジェクトの値にはそれぞれ名前がつけられる
    - その名前のことをプロパティという

- 式は計算であり、文は制御である
- 式を文として書くことができ、これを式文という
    - 式文は、その返す値は無視して副作用だけ実行する
